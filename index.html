01. Hosted Virtualization using VirtualBox and KVM

Hosted virtualization runs virtual machines (VMs) on top of an existing operating system (OS) using virtualization software like VirtualBox or VMware Workstation.
Example: You install VirtualBox on your Windows laptop and run Ubuntu Linux inside it as a virtual machine.
